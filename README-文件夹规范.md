# 《钢铁之心》中文原创项目 — 文件夹规范

## 根目录

`F:\html2\小说\钢铁之心-中文原创\`

## 正文存放（统一目录）

- **所有章节正文**放在根目录下的 **`正文\`** 文件夹中，**不再**为每一章单独建子文件夹。
- 单章文件命名建议：`第NNN章-简短标题-正文.md`（三位章号，便于排序）。
- 若同一章需要多版本，可在 `正文\` 内并列存放，例如：`第001章-灰烬之名-变体A.md`（文件名自洽即可）。

## 全书总细纲位置

- 见 `细纲\` 目录下按卷拆分的 Markdown 文件；**不要**在正文目录里复制几十万字总纲，避免重复维护。

正文创作前以 `细纲\` 为准；若正文与总细纲冲突，**以总细纲为准**并回写修订记录。
