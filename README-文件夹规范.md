# 《钢铁之心》中文原创项目 — 文件夹规范

## 根目录

`F:\html2\小说\钢铁之心-中文原创\`

## 每写一篇（每一章）的约定

- **每一章对应一个新文件夹**，命名格式：`第NNN章-简短标题\`（三位数字，便于排序）。
- 同一章的**不同变体**全部放在该章文件夹内，例如：
  - `变体A-冷硬叙事.txt` / `变体A-冷硬叙事.md`
  - `变体B-内心独白加重.txt`
  - `变体C-节奏更快-删减版.txt`
- 可选：`该章-细纲-备忘.md`（只对本章生效的局部调整，不替代全书总细纲）。

## 全书总细纲位置

- 见 `细纲\` 目录下按卷拆分的 Markdown 文件；**不要**在章节文件夹里复制几十万字总纲，避免重复维护。

## 版本说明

正文创作前以 `细纲\` 为准；若章节文件夹内变体与总细纲冲突，**以总细纲为准**并回写修订记录。
