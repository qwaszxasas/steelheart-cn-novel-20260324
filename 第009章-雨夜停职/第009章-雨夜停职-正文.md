# 第009章 雨夜停职

停职通知在雨夜送到。

红章盖得很正，字却冷：“即日起暂停沈某某现场取证职务，待调查结论。”

父亲把纸折好，夹进文件夹，没有骂一句。屋里只剩雨点敲窗和奶奶断断续续的咳嗽。

“这个月房租怎么办？”奶奶小声问。

父亲笑了笑：“先欠两天，能转开。”

沈砾听得牙根发酸。他知道那不是“能转开”，是“先别崩”。

第二天，父亲去跑临工，晚上才回来，鞋底全是泥。饭桌上，三个人谁都不提停职，像不提就没发生。

吃到一半，楼下突然一阵喧哗。有人在喊“快跑”“别点火”。

父亲立刻起身冲到窗边。

巷口便利店被一群人围住，起因是限量配给名单。有人说名单作假，有人说店主私卖，吵着吵着就动了手。一个黄毛拎着铁桶往卷帘门泼油，打火机“啪”地弹开。

“我下去。”父亲抓起外套。

“你都停职了！”沈砾拽住他。

“停的是岗位，不是人。”父亲挣开手，“看着奶奶，别下楼。”

父亲冲进雨里。

沈砾站在窗口，看见父亲挡在卷帘门前，张开双臂喝止人群。黄毛不听，抡起一根钢管砸过去。父亲侧身躲开，肩膀还是挨了一下，踉跄半步。

沈砾脑子“嗡”地一声。

他冲下楼时，雨已经把衣服打透。巷口乱成一锅，汽油味混着雨腥味，呛得人眼疼。

黄毛又举起钢管，这次直冲父亲后脑。

沈砾扑过去，一把攥住钢管中段。熟悉的灼痛从掌心炸开，他几乎咬碎后槽牙。

“给我松手！”黄毛大吼。

沈砾不松，反而盯死对方，呼吸一点点压稳。

钢管表面先起暗斑，再成片脱皮，最后“咔”地崩出一道裂口。黄毛还在发力，整根钢管当场断成两截，后半截反弹砸在他自己脚背上，疼得他跪地惨叫。

周围人全僵住。

“这小子是权柄者？”

“他手怎么回事？”

“跑！快跑！”

刚才还叫嚣的人一下散了大半。剩下几个人面面相觑，再没人敢往前凑。

父亲捂着肩站稳，先看卷帘门，再看沈砾的手，眼神里惊怒和担心搅在一起。

“谁让你下来的！”

“你要被打死了！”沈砾吼回去。

父亲被他噎住，半秒后低声骂了句“混账”，却没再吼，只把他往身后拉。

治安巡逻车终于姗姗来迟，照例先拍照、先维稳措辞。一个年轻警员看着地上的断钢管，皱眉问：“谁弄的？”

没人说话。

沈砾也没说。

回家路上，父子俩都湿透。楼道灯坏了，黑里只听见彼此呼吸。

“你今天这样，会被盯上。”父亲低声说。

“那我看你挨打？”

“我挨打和你暴露是两回事。”父亲停住脚步，“有能力，不等于有资格乱用。你每出一次手，都在给自己留痕。”

沈砾握紧拳，指缝发白：“那到底什么时候才有资格？”

父亲沉默很久，才说：“当你能承担后果的时候。”

进门后，父亲给他手掌上药。药水一碰伤口，疼得沈砾倒抽冷气。

父亲动作放轻，声音也低了：“今晚你救了人。这个我不否认。”

沈砾抬眼。

“但你得记住，”父亲看着他，“真正难的不是把钢管折断，是折断你自己心里那股‘我现在就能解决一切’的冲动。”

窗外雨还在下，钢城像一整块生锈铁皮。

夜深，沈砾在作业本背面写下：

“规则6：有能力，不等于有资格。资格=后果。”

写完他看着这行字，手指还在微抖。

他第一次意识到，爽点背后全是账。

这账，迟早有人来收。

（本章完）
