# 世界观与人物档案（原创设定）

## 时代与地理

- **地名怎么叫**：故事发生的主城，口语里大家都叫「**钢城**」（好记，像一座炼出来的城市）。红头文件、联播里有时写「钢垒都会区」——指同一片地方，读者不必纠结，当成「这座架空大城市」即可。
- **北郊旧展馆**：城北一座荒废的展览建筑，灾变前办过工业展，现在成了事故与清场常提的地标；文中也说「旧展馆」，避免单说「旧馆」让人摸不着头脑。
- **灾变后都市群**：旧国名已碎裂，现以钢城为核心——钢筋与玻璃构成的垂直迷宫，上空常年有灰黄色辐射尘与人工降雨。
- **权柄者（可改名定稿）**：灾变中觉醒的个体，能力各异，强度与「信念」「恐惧」「群体叙事」存在模糊关联（留作后期揭秘，细纲中只表现现象）。
- **秩序**：名义上的「临时联合治安署」实为权柄者寡头与企业的共治；底层街区自治、黑市与教会并存。

## 核心人物（暂定名，可全书统一替换）

| 角色 | 身份 | 叙事功能 |
|------|------|----------|
| **沈砾** | 主线视角（童年—成年双线可切） | 亲历「暴君」不可战胜的瞬间，执念与自我怀疑的载体 |
| **沈父** | 档案员/基层治安记录员（非权柄者） | 理性与规则的象征，其死亡触发主线命题：正义是否必须借暴力 |
| **「铸心」** | 钢城实际统治者之一，能力表现为近乎绝对物理免疫与区域金属操控 | 序章级反派脸谱下的多层伏笔（真名、弱点、为何留城不走） |
| **母亲线（可选）** | 早期失踪或改嫁，用于后期情感对照 | 避免人物过多时可在细纲中合并为「姑母」 |

## 主角能力（新版）

- **能力名（暂定）**：`锈触`。沈砾可在近距离触碰金属时，短时间加速其疲劳、氧化与脆化。
- **早期表现**：仅能影响小体积金属（门把、刀片、锁扣、棍头），且容易随情绪失控。
- **开篇爽点链**：前3章对应“觉醒 -> 反击 -> 稳控压制”，让读者快速建立期待。
- **成长代价**：能力越强，反噬越重（体温骤降、手指裂痛、短时眩晕）。

## 语言风格（面向华人读者）

- 句法偏**现代汉语书面语**，适度保留口语节奏；避免翻译腔（少用「正在」「进行」堆砌）。
- 专有名词**短、利、可记**：机构名两到四字，能力名避免英文直译感。
- 文化锚点：可自然出现**年夜饭、学籍、户籍、城中村、通勤**等本土生活细节，与架空都市形成对照，**不**写成现实城市影射。
